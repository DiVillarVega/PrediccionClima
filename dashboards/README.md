# Dashboard

Aquí irá el dashboard interactivo. Recomendación para la siguiente etapa:
- Streamlit para prototipo rápido.
- Plotly para gráficos interactivos.
- Filtros: ciudad, rango de fechas, variable climática.
- KPI de producto: predicción de temperatura máxima, probabilidad/riesgo de lluvia y alerta operativa.
